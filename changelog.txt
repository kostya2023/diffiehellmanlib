Changelog

    Ver 1.0
         - Дата: 10.01.2025
         - Поддержка библиотеки для архитектуры Windows x64. 

    Ver 1.1
         - Дата: 10.01.2025
         - Добавлена поддержка архитектуры Windows x86.
         - Интеграция с OpenSSL для обеих архитектур.
         - Обновлён Python-враппер:
             - В зависимости от архитектуры автоматически подгружаются соответствующие библиотеки (x86 или x64).

    Ver 1.2
         - Дата: 10.01.2025
         - Добавлены библиотеки для x64 и x86 Linux.

    Ver 1.3
         - Дата: 10.01.2025
         - Добавлены библиотеки для ARMv7 и ARM64.

    Ver 1.4
         - Дата: 10.01.2025
         - Добавлен Python-враппер для Linux:
             - Автоматически подгружает нужную библиотеку (x86, x64, ARMv7, ARM64) в зависимости от архитектуры.

    Ver 1.5 - "Крупный апдейт."
         - Дата: 10.01.2025
         - Функции переименованы в более понятные имена, мелкие поправки в функциях, и т.д.
             - Функция generate_p_g_a
                  - Переименована в generate_p_g 
                  - Аргумент bits стал int.
                  - В dll и so файлах функцию менять не стал, просто убрал во врапере вывод a, освобождение оставил чтобы не было утечек памяти.

             - Функция generate_b
                  - Переименована в generate_a_or_b
          
             - Функция generate_A
                  - Переименована в generate_A_or_B
          
         - Изменена немного логика работы generate_shared_key
             - Функции generate_shared_key и hash_shared_key поменяны местами.
             - Вывод generate_shared_key сразу хеширован, что уменьшает количество действий в основном коде.

         - Добавлен мануал или же документация.

         - Добавлена python библиотека (можно установить pip install diffie_hellman_lib) подробнее на https://pypi.org/project/diffie-hellman-lib/
